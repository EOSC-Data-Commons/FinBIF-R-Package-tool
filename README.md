# FinBIF-R-Package-tool

This jupyter notebook is quick demonstration of how to build a species distribution model (SDM) for Eurasian Red Squirrel (Sciurus vulgaris) using data from Finnish Biodiversity Information Facility (FinBIF). 

## Getting Started
The simplest way to run the notebook is to use an online service like Google colab. You can also run the notebook locally using Jupyter Notebook or Jupyter Lab.

## Requirements
You need FinBIF API access token. You can request it by following the instructions of the notebook. Also you need R language and related libraries specified on the file.
